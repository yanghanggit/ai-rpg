# 桃花源村民角色图集 概念美术提示词

## 项目信息

- **用途**: 村民角色图集 (Character Sprite Sheet)
- **尺寸**: 1024 x 1024 (正方形 1:1)
- **生成模型**: Gemini 2.5 Flash Image (nano-banana) / Replicate API
- **模型特点**: 支持中文提示词，原生 1K 分辨率输出
- **艺术风格**: 16位像素游戏风格（八方旅人风格）
- **场景类型**: 4x4网格角色立绘图集（16个村民全身像，每格256x256像素）
- **支持语言**: 中文 (zh-CN)
