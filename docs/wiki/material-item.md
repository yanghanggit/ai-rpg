# 材料系统（MaterialItem）

---

## 定位

`MaterialItem` 是**不可直接使用**的原料类道具，整个生命周期仅驻留于全局储物箱，不参与战斗，也不携带任何属性词缀。其唯一设计用途是在家园工坊中被消耗，经由 LLM 推理合成为 `ConsumableItem`。

这种设计将材料与消耗品的"生产-消费"关系清晰分离：材料是工坊的原料入口，合成是其唯一出口。

---

## 与工坊系统的交互

工坊合成由 `WorkshopComponent` 实体作为 LLM agent 驱动。材料的描述文本是 LLM 推断合成结果的**唯一语义线索**——材料描述越具体，合成出的消耗品词缀与能力越具个性化。这正是材料描述字段存在的核心设计意图，也解释了为何材料与消耗品均采用带前缀的命名约定（`材料.XXX`）以维持语义清晰度。

合成完成后，所用材料从储物箱中扣减；材料归零则被移除，合成的 `ConsumableItem` 取而代之进入储物箱。
