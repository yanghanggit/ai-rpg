## 🎯 设计意图

### 核心元素

- **桃花源秘境**: 云雾缭绕的武陵山谷
- **秦汉建筑**: 飞檐翘角的古老村落
- **神话异兽**: 《山海经》风格的神秘巨兽剪影
- **景深层次**: 前景桃花 → 中景建筑 → 远景群山

### 艺术风格

- **水墨晕染**: 传统中国水墨画技法
- **留白美学**: 适度留白，便于叠加游戏 Logo
- **色彩克制**: 墨色为主，淡粉色桃花点缀
- **意境营造**: 神秘、幽静、世外秘境

### Launch Page 适配

- ✅ 竖屏构图 (9:16)
- ✅ 无人物角色
- ✅ 适度留白（便于上层叠加标题/Logo）
- ✅ 强烈的游戏世界观识别度

## ⚠️ 避免生成UI元素的方法论

### 问题诊断

当提示词中包含"游戏启动页面"、"Launch Screen"等描述时，AI 模型容易误判，主动添加：

- 游戏标题文字
- START GAME 按钮
- 装饰性边框
- Logo 或图标

### 解决方案

**正向提示词策略：**

1. ❌ **避免使用**：
   - "游戏启动页面设计"
   - "Launch Screen"
   - "游戏界面"
   - "UI设计"

2. ✅ **改用描述**：
   - "纯背景插画艺术作品"
   - "概念美术插画"
   - "背景原画"

3. ✅ **明确声明**（在提示词末尾添加）：
   - "纯粹的背景艺术，无任何文字、标题、按钮或界面元素"
   - "不包含任何 UI 元素"

**负向提示词策略：**

1. ✅ **强化排除**（需明确列举）：
   - 文字类："文字，汉字，英文字母，标题，游戏标题"
   - UI 元素："按钮，UI界面，游戏界面，START，PLAY"
   - 装饰元素："框架，边框，装饰框，红框，logo，标志"

2. ✅ **经验法则**：
   - 负向提示词需要**具体且详尽**
   - 如果生成出现了某个UI元素，立即将其添加到负向提示词中
   - 常见触发词：START, PLAY, MENU, 开始游戏 等

### 实测案例

| 提示词版本 | 结果 |
| ----------- | ------ |
| "手机游戏启动页面设计" | ❌ 生成了游戏标题和 START GAME 按钮 |
| "纯背景插画艺术作品" + 强化负向词 | ✅ 纯净背景，无UI元素 |

## 📝 提示词解析

### 关键术语

- **"水墨晕染技法"**: 比英文 "ink gradients" 更精准地传达中国画技法
- **"飞檐翘角"**: 精确描述秦汉建筑特征
- **"层峦叠嶂"**: 中国山水画专用术语
- **"留白处理"**: 水墨画核心美学原则
- **"世外秘境"**: 准确传达《桃花源记》的文学意境

### 游戏设定对应

| 设定元素 | 提示词体现 |
| --------- | ----------- |
| 武陵山脉秘境 | "武陵山脉深处的桃花源秘境" |
| 双重失落文明 | "秦汉风格的古老村落" + "《山海经》神话异兽" |
| 避秦之民 | "飞檐翘角的古建筑群" |
| 古先民遗迹 | "神秘巨兽剪影" |
| 生态层级 | 多层次景深：桃花、建筑、山峰、异兽 |

## 🔄 迭代优化建议

如果第一次生成效果不满意，可以调整：

1. **水墨感不足**: 增加 "宣纸质感"、"干湿浓淡变化"、"飞白技法"
2. **异兽不够突出**: 强化 "石狮体积感"、"巨兽轮廓清晰度"
3. **桃花过于明显**: 调整为 "若隐若现的桃花点缀"
4. **留白不够**: 强调 "上部留白供标题使用"
5. **色彩过艳**: 增加 "克制的色彩"、"素雅色调"
6. **出现UI元素**: 检查正向提示词是否有触发词（如"游戏界面"），强化负向提示词（添加具体UI元素名称）
