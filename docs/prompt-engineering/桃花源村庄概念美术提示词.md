## 🎯 设计意图

### 核心元素

- **秦汉建筑群**: 飞檐翘角、青瓦白墙、木质结构
- **桃花环绕**: 盛开的桃花林包围村庄，营造梦幻氛围
- **田园景观**: 石桥流水、良田阡陌、竹林桑树
- **生活细节**: 红灯笼、陶罐、农具、炊烟（但无人物）
- **景深层次**: 前景建筑细节 → 中景村落全貌 → 背景虚化群山

### 艺术风格

- **水墨晕染**: 传统中国水墨画技法，强调建筑质感
- **留白美学**: 背景虚化留白，突出村落主体
- **色彩克制**: 墨色为主，淡粉色桃花，少量暖色点缀（红灯笼）
- **意境营造**: 宁静、古朴、田园诗意、世外桃源

### 场景适配

- ✅ 竖屏构图 (9:16)
- ✅ 无人物角色（强调空灵感）
- ✅ 中近景特写（便于展示建筑细节）
- ✅ 强烈的"避秦之民"古朴村落氛围

## ⚠️ 避免生成UI元素的方法论

### 问题诊断

当提示词中包含"游戏场景"、"村庄界面"等描述时，AI 模型容易误判，主动添加：

- 游戏标题文字
- 界面按钮
- 装饰性边框
- 小地图图标

### 解决方案

**正向提示词策略：**

1. ❌ **避免使用**：
   - "游戏场景设计"
   - "村庄界面"
   - "游戏地图"
   - "UI设计"

2. ✅ **改用描述**：
   - "纯背景插画艺术作品"
   - "村落特写"
   - "场景概念美术"

3. ✅ **明确声明**（在提示词末尾添加）：
   - "纯粹的背景艺术，无任何文字、标题、按钮或界面元素"

**负向提示词策略：**

1. ✅ **强化排除**（需明确列举）：
   - 文字类："文字，汉字，英文字母，标题"
   - UI 元素："按钮，UI界面，游戏界面，START，PLAY"
   - 装饰元素："框架，边框，装饰框，红框，logo，标志"

2. ✅ **经验法则**：
   - 负向提示词需要**具体且详尽**
   - 如果生成出现了某个UI元素，立即将其添加到负向提示词中

## 📝 提示词解析

### 关键术语

- **"秦汉风格建筑"**: 精确指定历史时期的建筑特征
- **"飞檐翘角，青瓦白墙"**: 传统建筑的标志性元素
- **"阡陌交通"**: 出自《桃花源记》原文，指田间小路交错
- **"炊烟袅袅"**: 营造生活气息但不出现人物
- **"世外桃源"**: 准确传达游戏核心意境

### 游戏设定对应

| 设定元素 | 提示词体现 |
| --------- | ----------- |
| 避秦之民村落 | "秦汉风格建筑群错落有致" |
| 桃花源环境 | "被盛开的桃花林环绕" |
| 田园生活 | "良田阡陌，竹林婆娑，桑树点缀" |
| 古朴氛围 | "木质门窗古朴典雅"、"石板小径" |
| 生活痕迹 | "红灯笼"、"陶罐"、"农具"、"炊烟" |
| 无人世界 | "画面中不出现人物" |

## 🏘️ 村庄场景特有要素

### 建筑元素细化

- **屋顶**: 青色瓦片，飞檐翘角
- **墙体**: 白墙或土墙，斑驳有年代感
- **门窗**: 木质结构，朴素无华
- **装饰**: 红灯笼（暖色点缀），避免过度装饰

### 自然元素细化

- **桃花**: 盛开状态，花瓣飘落
- **溪流**: 清澈见底，石桥横跨
- **植被**: 竹林、桑树、柳树（《桃花源记》元素）
- **田地**: 整齐划分，展现农耕文明

### 生活痕迹（无人物）

- **炊烟**: 从屋顶升起，营造生活气息
- **晾晒物**: 可选，如晾晒的衣物或农作物
- **工具**: 农具、陶罐、竹篮等静物
- **动物**: 可选，如鸡鸭（但需谨慎，可能触发"角色"）

## 🔄 迭代优化建议

如果第一次生成效果不满意，可以调整：

1. **建筑不够突出**: 增加 "建筑细节精细刻画"、"木质纹理清晰"
2. **桃花过于喧宾夺主**: 调整为 "桃花点缀其间"、"淡粉色花瓣若隐若现"
3. **缺乏生活气息**: 强化 "炊烟"、"红灯笼"、"院落中的生活痕迹"
4. **构图过于平面**: 增加 "石板小径引导视线"、"建筑错落有致的纵深感"
5. **色彩过于单调**: 适度增加 "暖色调的红灯笼作为视觉焦点"
6. **背景过于复杂**: 强调 "背景群山虚化留白"、"突出村落主体"
7. **出现人物**: 检查是否有"村民"、"生活场景"等触发词，强化负向提示词
8. **出现UI元素**: 参考"避免生成UI元素的方法论"章节
