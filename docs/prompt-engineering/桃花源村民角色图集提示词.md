# 桃花源村民角色图集 概念美术提示词

## 项目信息

- **用途**: 村民角色图集 (Character Sprite Sheet)
- **尺寸**: 1024 x 1024 (正方形 1:1)
- **生成模型**: Gemini 2.5 Flash Image (nano-banana) / Replicate API
- **模型特点**: 支持中文提示词，原生 1K 分辨率输出
- **艺术风格**: 16位像素游戏风格（八方旅人风格）
- **场景类型**: 4x3网格角色立绘图集（12个村民全身像）
- **支持语言**: 中文 (zh-CN)
- **创建日期**: 2026年1月13日

## 📝 设计要点

### 视觉风格

- **像素风格**: 16位复古游戏美术，参考《八方旅人》系列
- **头身比例**: 小头身（约1:3或1:4），面部细节极简
- **色板限制**: 使用有限色板，经典像素艺术技法

### 布局要求

- **网格排列**: 严格4行3列，共12个格子
- **背景处理**: 纯白色填充（#FFFFFF）
- **间距对齐**: 等距排列，每个角色在格子内居中

### 角色特点

- **服饰风格**: 秦汉朴素服装，避秦之民风格
- **职业识别**: 通过服装颜色、道具装备区分职业
- **面部处理**: 极简化，看不清五官细节

### 禁止元素

- 无现代元素、UI界面、文字标题
- 无角色重叠、拥挤构图
- 无场景背景、边框装饰
