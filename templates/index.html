内部研发用测试网页

处理指令:

游戏信息

可用指令


# 退出游戏

/quit

# 创建游戏房间

/create

# 加入已经存在的房间

/join @127.0.0.1

# 选择角色

/pickactor @无名的复活者

/pickactor @教廷密使

# 对谁使用某个道具

/useprop @禁言铁棺>腐朽的匕首

# 攻击目标NPC, 名字为Name?

/attack ‘Name?’

# 离开当前场景，去往Name?的场景

/leave ‘Name?’

# 在当前场景内广播内容。场景内所有NPC都能听见

/broadcast ‘说的内容’

# 对当前场景内的目标说话

/speak ‘@对谁>说话内容’

# 对当前场景内的目标低语

/whisper ‘@对谁>说话内容’

# 在当前场景内搜索叫‘Name?’的道具

/search ‘Name?’

# 在不知道去往哪里的情况下，‘跳出’当前场景，如果当前场景没有连接场景则会失败。

/portalstep

# 感知当前场景内有哪些人？事？道具？

/perception

# 盗取对当前场景内的目标的道具

/steal ‘@对谁>盗取的道具名字’

# 将我身上的道具交给目标

/trade ‘@对谁>我的道具的名字’

# 查看我身上有哪些道具？

/checkstatus
