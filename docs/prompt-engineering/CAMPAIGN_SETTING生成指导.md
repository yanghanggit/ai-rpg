# CAMPAIGN_SETTING 生成指导

## 任务目标

生成世界观设定（90-140字），作为 System Prompt 的一部分。

**定位**：桌游 GM 开场白 + 百科目录页

- 提供世界框架和索引
- 不包含当前剧情或玩家角色
- 详细信息通过 RAG 动态补充

---

## 结构模板（3个模块）

### 模块 1：世界定位（20-30字）

```text
[世界名]: [类型标签] + [核心概念] + [地理特征] + [象征意义]
```

### 模块 2：空间索引（40-60字）

```text
主要区域: [区域A](特征)、[区域B](特征)
关键地标: [地标名] + [其意义]
```

### 模块 3：势力索引（30-50字）

```text
主要势力: [势力A]、[势力B]、[势力C] [总体定位]
次要组织: [组织列表] [行为模式]
```

---

## 约束规则

### ✅ 必须

- 仅列名称，不展开介绍
- 使用复合句和括号压缩信息
- 使用"目录式"语言
- 总字数 90-140 字

### ❌ 禁止

- 展开势力职能描述（留给 RAG）
- 描述"当前状态"或"正在发生"的事件（运行时注入）
- 包含玩家定位或任务目标（角色设定）
- 过度渲染形容词

---

## 标准示例

```text
新奥拉西斯: 人类、精灵、兽人三族联盟守护的圆形圣城，
坐落于翠绿河谷中心，被誉为「希望熔炉」。
城市分两层: 外环贫民区「尘烟裂谷」(工坊密集)、
内环富人区「耀光宝冠」(环绕封印之塔)。
封印之塔供奉圣剑「晨曦之刃」，封印魔王玛拉凯斯。
三大势力维系秩序: 星辉学院、银叶秘苑、砺石角斗场。
五大帮派——铁齿帮、暗影信使、瘴气拾荒者、
裂隙医师协会、遗迹掘客团，通过「裂谷议会」活动于尘烟裂谷。
```

---

## 检查清单

- [ ] 字数 90-140 字，Token 约 300-450
- [ ] 包含世界定位、空间索引、势力索引
- [ ] 仅列名称，无详细介绍
- [ ] 无当前冲突或玩家定位
- [ ] 使用复合句压缩信息

---
